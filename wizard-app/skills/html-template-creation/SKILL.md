# Template Creation

## When to use this skill
Use this skill when creating a new HTML template in an ionic app.

## how to wrap the template
- always wrap each template in a container div
- assign the container div to a class named <component-name>-container
- tables and sub-components should always be put in a container

## which html elements to use
- use div-elements for each text and button elements
- no use of <br>, <p>, <h1>, <button> unless said in prompt

## style assignment
- each html element should have a class assigned
- containers will have class specified in the wrapper section
- text parts should have either a title or a description class, depending on the context
- do not use any semantic html elements like <i> or <strong>

## ion element usage
- do not use any ionic elements unless specified in prompt, rely on classic html/angular syntax
